# fin_float

`fin_float` 是一款轻量、极简的桌面悬浮盯盘工具。它能够以极简的图形或数字方式，在屏幕边缘或任意位置悬浮显示指定标的的今日波动，助你从容掌握市场动态，安心专注手头工作。

---

## ✨ 核心特性

### 极简视觉
支持精简图形或纯数字模式，不占用宝贵的屏幕空间

## 🛠️ 安装与运行

### 快速开始
  1 -> 点击下方任一传送门，去发布页面下载最新版本

👉 **[下载链接/GitHub](https://github.com/IceFRain/fin-float/releases)**

👉 **[下载链接/Gitee](https://gitee.com/IceFRain/fin-float/releases)**

  2 -> 解压下载好的压缩包，双击 **`fin-float`** 运行

  3 -> 双击屏幕中显示的小斜杠打开设置界面
<br/><img width="12" height="21" alt="image" src="https://github.com/user-attachments/assets/148a28fd-c132-440e-ae5b-266dc9240608" />

  4 -> 添加监视标的，配置喜好后保存即可

  注：①刷新间隔填0时，软件将不会自动刷新，可右键点击悬浮窗手动刷新 ②右键点击悬浮窗选择“退出”即可退出软件

### 手动添加开机自启

  软件添加开机自启配置项，如需开机自启，请创建 **`fin-float`** 的快捷方式，放置到 **`shell:startup`** 文件夹中即可
